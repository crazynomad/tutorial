# Nano Banana 比例约束与变通

## 支持的比例

Nano Banana 2 **只支持**：

| 比例 | 像素参考 | 用途 |
|---|---|---|
| 16:9 | 1920×1080 | HF 全幅背景、封面、封底 |
| 4:3 | 1024×768 | 经典幻灯比例（本 skill 不用） |
| 1:1 | 1024×1024 | 图标、DOT 圆点、社交封面 |
| 3:4 | 768×1024 | R34 / L34 竖图 |
| 9:16 | 1080×1920 | 手机竖版、长图 |

**不支持**：21:9 超宽、2.39:1 电影宽银幕、自定义比例。

## 21:9 超宽需求的变通

如果版式真的需要 21:9（比如全屏 banner、电影感开场），三条路：

### 路线 1（推荐）：16:9 + 左右裁切
生成 16:9，在 HTML 里用 `object-fit: cover` + 定位裁掉上下或左右。视觉上等效超宽。

```css
.banner-21-9 {
  width: 720pt; height: 308pt; /* 21:9 */
  background: url('slide.png') center/cover no-repeat;
  /* 原图 16:9，center 裁切上下 */
}
```

### 路线 2：双图拼接
生成两张 3:4 或两张 16:9，在 HTML 里并排。适合对比式构图（"before / after"、"A / B"）。

### 路线 3：HF 铺底 + 前景元素
把 21:9 的"主体"单独生成 3:4 或 1:1，背景用 16:9 全幅铺底，前景合成到安全区。

## 比例-Layout 对照

| Layout | Nano Banana 比例 | 注释 |
|---|---|---|
| HF (Hero-Fill) | 16:9 | 部署加 62% 暗罩 |
| R34 / L34 | 3:4 | top/bottom 各 15% 留白 |
| DOT (圆点/图标) | 1:1 或复用 3:4 中心裁切 | 不必单独生成 |

## 生成时的质量守则

- **一次生成多张**（4~8 张）挑优，Nano Banana 有随机性
- **关键 prompt 不改色彩 token** —— 同一系列色彩要视觉连贯
- **生成完检查**：有没有 AI 水印 / 错误文字 / 多指 / 畸变 → 有就重跑
- **存档**：保留源图 + 备份到 `images/backup/`，方便回滚
