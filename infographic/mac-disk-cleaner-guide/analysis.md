## Main Topic
这是一期围绕 Claude Code + Skills 的实战教程，演示如何用 3 个免费 Skills（磁盘清理、文件整理、文档提炼）在 Mac 上释放 120G 磁盘空间并构建个人知识库。核心信息是"0 代码、说人话就能用"。

## Learning Objectives
1. 理解"瘦身 → 收纳 → 提炼"三阶段工作流及每阶段的工具、操作和产出
2. 掌握每个 Skill 背后的核心技术组件（Mole / macOS 原生能力 / MarkItDown+Ollama）
3. 了解 Claude Code vs 网页版 Claude 的本质区别，以及如何安装和获取 Skills

## Target Audience
- **Knowledge Level**: 初级（非程序员 Mac 用户）
- **Context**: 看完视频后想对照操作，或独立阅读的图文攻略
- **Expectations**: 清晰的操作步骤、资源链接、工具对比

## Content Type Analysis
- **Data Structure**: 三阶段线性流程（瘦身→收纳→提炼），每阶段含子步骤；附带对比表（Claude vs Claude Code）和资源列表
- **Key Relationships**: Skills 是 Claude Code 的"知识胶囊"；每个 Skill 背后包装了一个开源工具；三阶段递进（从简单到硬核）
- **Visual Opportunities**: 三阶段流程图、数据对比（31G→137G）、技术架构分解、工具生态图

## Design Instructions (from user input)
- 用户希望做"详尽的游戏攻略"风格
- 需要汇总所有资源和知识点
- 信息密度高，适合长图/竖图
