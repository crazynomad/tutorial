## Main Topic
通过"书架 vs 书单"的生活类比，帮助 macOS 用户理解智能文件夹与传统文件夹的本质区别。

## Learning Objectives
After viewing this infographic, the viewer should understand:
1. 传统文件夹是物理容器，智能文件夹是保存的搜索条件
2. 两者在文件位置、删除行为、多处出现等维度的具体差异
3. "书架 vs 书单"类比，一句话记住核心区别

## Target Audience
- **Knowledge Level**: Beginner — macOS 普通用户
- **Context**: 视频教程配图，需要直观易懂
- **Expectations**: 看完立刻理解智能文件夹是什么、有什么好处
