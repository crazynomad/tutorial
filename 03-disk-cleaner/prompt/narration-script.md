## Scene 03: 原理讲解

> 刚才我们演示了瘦身、收纳、提炼三个 Skill。接下来我们逐个拆解一下，它们背后到底是怎么运作的。

### 3.1 瘦身——磁盘清理 Skill

> 先说磁盘清理。它的核心引擎是 tw93 开发的 Mole，GitHub 上三万多 Star 的开源项目。Mole 有 clean、analyze、purge 等一系列命令，知道 macOS 上哪些缓存可以安全删除、哪些绝对不能碰。

*(画面：MoleFeatures 动画)*

> 但 Mole 本身是个命令行工具，你得知道怎么装、怎么传参数、怎么配白名单。Skill 做的事情，就是把这些操作翻译成人话——你说"磁盘空间不够了"，Skill 就帮你调用 Mole 的扫描、生成报告、执行清理，全程不用碰一行命令。

### 3.2 收纳——文件整理 Skill

> 再说文件整理。这个 Skill 背后用的是 macOS 自带的能力——Finder 智能文件夹、文件元数据、Spotlight 索引。这些都是你电脑上本来就有的功能，只是大多数人不知道怎么用。

> Skill 的价值在于：它先用脚本扫描你的文件类型和大小分布，做出分析报告；然后根据分析结果，自动创建智能文件夹、按规则归类、批量清理。整个流程串起来，就变成了一句话能搞定的事。

### 3.3 提炼——文档智能 Skill

> 最后是文档智能，这个最硬核。它背后有三个关键组件。

> 第一个是格式转换。PPT、PDF、Word 这些文件，先通过开源工具转成纯文本的 Markdown，这样 AI 才能读懂内容。

> 第二个是本地摘要。用 Ollama 跑一个本地 AI 模型，给每份文档生成摘要。注意，这一步完全在你自己电脑上跑，文件不会上传到任何地方。

> 第三个是智能分类。根据摘要内容，从主题、用途、客户三个维度自动归类，还能用 AI 给文件重新起一个看得懂的名字。

*(画面：DocMindmapFeatures 动画)*

### 3.4 Skill 的本质

> 所以你看，这三个 Skill 背后用的都是已有的工具——Mole、macOS 原生能力、Ollama、开源转换库。但这些工具，普通人都不会用。

*(画面：SkillArchitecture 动画)*

> 还记得上期说的嘛，Skill 就是一本给 AI 准备的入职手册。我们说人话，Skill 把它翻译成底层工具的命令。中间这层翻译，就是 Claude Code 在做的事。

*(画面：OpenSourceEco 动画)*

> 这就是 Claude Code 真正的魔力。它把大神的开源工具，通过 Skill 包装起来，让普通人也能一句话调用。你不需要会写代码，只需要能清楚地描述你要做什么。

## Scene 04: Claude.ai vs Claude Code

### 4.1 失忆症

*(画面：分屏对比动画)*

> 有小伙伴可能会问：网页版 Claude 不也挺强的吗？为什么非要用 Claude Code？

> 三个字：够不到。

> 网页版有个致命的问题——失忆症。每次新开对话，它完全不记得你是谁，你的文件在哪。你每次都得把文件拖进去上传。

> 而 Claude Code 运行在我们自己的电脑上，你的文件就是它的上下文。不需要上传，不需要复制粘贴。

---

### 4.2 动嘴 vs 动手

> 第二个区别。网页版只能告诉你——"建议你打开 Finder，找到资源库，手动删除浏览器缓存"。

> Claude Code 呢？直接帮你删了。106 个 G，实打实地释放。

> 一个 Talking，一个 Doing。一个动嘴，一个动手。差距就在这。

---

### 4.3 系统升级

> 更关键的是第三个区别。

> 装上 Claude Code，就像给你的操作系统加了一层 AI。而每装一个 Skill，就相当于给这个系统打了一次升级补丁。

> Skill 装一次就够了。下次想清理磁盘？打开终端，说一句话，搞定。

> 网页版呢？你得重新开对话，重新解释你要什么，重新上传文件。每次都从头来。

> 这不是一次性的对话，而是在给你的电脑做系统升级。每安装一个 Skill，你的 AI 助手就多学会一项本事。装得越多，系统越强。

> 说到这，最近有个叫 OpenClaw 的东西特别火，对吧？GitHub 上十万多 Star。它能自己操作你的电脑、自己点鼠标、自己切应用，你在微信或者 Telegram 上发句话，它就帮你把事办了。看着特别科幻。

*(画面：OpenClaw 官网介绍动画——龙虾 Logo + "The AI that actually does things" + 多平台集成展示)*

> 但你知道吗，OpenClaw 背后的核心支撑之一，就是 Claude Code。只不过 OpenClaw 更加激进——它直接接管了你的屏幕和鼠标。

> 而我们今天用的 Claude Code + Skill 的方式，其实是同一条路线上更稳、更可控的玩法。你清楚地知道它在做什么，每一步都经过你的确认。先把这条路走通了，将来 OpenClaw 这类工具真正成熟的时候，你已经准备好了。
