# Software 软件目录

本文件夹包含制作 AI 工具视频介绍所需的软件和资源。

> **注意**: 本文件夹内的二进制文件和视频文件不会上传到 Git 仓库，仅作本地使用。

## 📦 软件清单

### 屏幕录制软件

#### Screen Studio
- **文件名**: `Screen Studio 3.5.1-4051 Apple Silicon.dmg`
- **版本**: 3.5.1-4051
- **平台**: Apple Silicon (M系列芯片)
- **大小**: ~366 MB
- **用途**: 专业屏幕录制软件，适用于创建高质量的演示视频

#### ScreenFlow
- **文件名**: `ScreenFlow_10_5_1_TNT.iso`
- **版本**: 10.5.1
- **大小**: ~99 MB
- **用途**: Mac 平台专业视频编辑和屏幕录制软件

### 教程资源

#### ScreenFlow 教程
- **文件夹**: `ScreenFlow教程/`
- **内容**: ScreenFlow 使用教程和相关学习资料

## 📝 使用说明

1. 这些软件用于录制和编辑 AI 工具的演示视频
2. 二进制文件（.dmg, .iso）已被 .gitignore 排除，不会提交到仓库
3. 建议将软件安装在本地后使用，保持此文件夹作为安装包备份

## 🔧 安装指南

### Screen Studio 安装
```bash
# 打开 dmg 文件
open "Screen Studio 3.5.1-4051 Apple Silicon.dmg"
# 将应用拖动到 Applications 文件夹
```

### ScreenFlow 安装
```bash
# 挂载 iso 文件
open "ScreenFlow_10_5_1_TNT.iso"
# 按照提示完成安装
```
